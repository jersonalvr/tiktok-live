Endpoint

You can connect to the [WebSocket](https://developer.mozilla.org/en-US/docs/Web/API/WebSocket) endpoint using the following URL:

`ws://localhost:21213/`

### Events

The following events are supported:

- `liveStatus`: Indicates if the user is currently live. Data: `{ isLive: boolean }`
- `member`: User joined the live room
- `roomUser`: Room user sequence message
- `follow`: User followed the creator
- `like`: User liked the live
- `chat`: User sent a chat message
- `share`: User shared the live
- `gift`: User sent a gift
- `social`: Social message
- `emote`: Emote chat message
- `envelope`: Envelope message
- `questionNew`: New question
- `linkMicBattle`: Link mic battle
- `linkMicArmies`: Link mic armies
- `liveIntro`: Live intro message
- `subscribe`: User subscribed
